# Projeto CSS 1 - Botões Animados

## 📋 Descrição
5 botões com efeitos de hover animados usando apenas CSS.

## 🎯 Conceitos Aprendidos
- Pseudo-elementos (::before, ::after)
- Transições CSS
- Animações com @keyframes
- Transformações 2D/3D
- Gradientes CSS
- Grid Layout
- Responsividade

## 🚀 Como Usar
1. Abra `index.html` no navegador
2. Passe o mouse sobre os botões para ver os efeitos
3. Clique nos botões para ver o feedback

## 📱 Responsivo
Funciona em desktop, tablet e mobile.
