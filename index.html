Como Funciona

Este projeto explora as possibilidades da pseudo-classe :hover e transições CSS para criar botões interativos e visualmente atraentes.

Conceitos Principais

  • :hover: Pseudo-classe que aplica estilos quando o mouse está sobre o elemento
  • transition: Propriedade que suaviza mudanças de estilo
  • transform: Para movimentos, rotações e escalas
  • box-shadow: Para criar profundidade e efeitos de luz
  • pseudo-elementos: ::before e ::after para elementos adicionais

Estrutura Básica de um Botão

<!-- HTML -->
<button class="btn">
  Meu Botão
</button>

/* CSS */
.btn {
  padding: 12px 24px;
  background: #3498db;
  color: white;
  border: none;
  border-radius: 5px;
  cursor: pointer;
  transition: all 0.3s ease;
}

.btn:hover {
  background: #2980b9;
  transform: translateY(-2px);
}

Exemplo: Botão com Efeito de Preenchimento

.btn-fill {
  position: relative;
  overflow: hidden;
  z-index: 1;
}

.btn-fill::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: #e74c3c;
  transition: left 0.3s ease;
  z-index: -1;
}

.btn-fill:hover::before {
  left: 0;
}

Boas Práticas

  • Use transições suaves (0.2s - 0.4s)
  • Mantenha a usabilidade - não exagere nos efeitos
  • Teste em diferentes dispositivos
  • Considere a acessibilidade (contraste de cores)
